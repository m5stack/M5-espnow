# M5-espnow

M5-espnow可以让通信更加方便的使用。

M5-espnow是基于乐鑫ESPNOW做的二次封装的库，使得在M5产品上使用更容易简单。(注意：ESPNOW规定每次发送最多不超过250个字节)
